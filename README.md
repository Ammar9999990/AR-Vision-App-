# 🤖 AI Chatbot Application (.NET Windows Forms)

## 📌 Project Overview
This project is an AI-based chatbot developed using Windows Forms in .NET. The application allows users to interact with an AI model and receive intelligent responses in real-time.

The chatbot uses Google's Gemini API (Gemini 2.5 Flash) to process user input and generate responses.

---

## 🎯 Features
- User-friendly Windows Forms interface
- Real-time AI responses
- Integration with Gemini API
- Error handling for API requests
- Continuous chat interaction

---

## 🛠️ Technologies Used
- C#
- .NET Framework (Windows Forms)
- Newtonsoft.Json
- HttpClient
- Google Gemini API

---

## ⚙️ How It Works
1. User enters a message in the input box
2. Application sends request to Gemini API
3. API processes input using AI model
4. Response is returned in JSON format
5. Application displays the response in chat window
